# AI Placement Platform

# Animation Guidelines

Document Version: 1.0

Status: Frozen

Sprint: Frontend Sprint F1

---

# Purpose

Defines animation standards used throughout the application.

Animations should improve usability, provide feedback, and reinforce progress without distracting users.

---

# Animation Principles

- Purposeful
- Smooth
- Consistent
- Fast
- Minimal

Animations must never delay user interaction.

---

# Motion Philosophy

Motion communicates

- Progress
- Navigation
- Success
- Feedback
- Context

Motion should never exist purely for decoration.

---

# Motion Categories

## Page Transition

Fade + Slide

Duration

200–300ms

---

## Card Hover

Slight elevation

Scale

1.02

---

## Button Interaction

Hover

Slight lift

Click

Small compression

---

## Dialog

Fade

Scale

---

## Drawer

Slide from left

Mobile bottom sheet slides upward.

---

## Career Orbit™

Animated

- Node glow
- Progress updates
- Hover effects
- Orbit expansion
- Smooth transitions

Career Orbit™ should feel alive.

---

## Charts

Animate once on load.

Do not repeatedly animate data.

---

## Loading

Skeleton loaders

Progress indicators

Subtle shimmer

---

## Success Feedback

Examples

- Resume Improved
- Interview Completed
- Career Orbit Increased

Use subtle celebration animations.

---

## Error Feedback

Shake animation

Fade message

Retry highlight

---

## Notification

Slide in

Auto dismiss

Manual close

---

# Duration Standards

Fast

150ms

Normal

250ms

Slow

400ms

Never exceed 500ms.

---

# Easing

Standard easing throughout the application.

Avoid inconsistent animation timing.

---

# Reduced Motion

Respect user accessibility preferences.

Disable non-essential animations.

Keep functional transitions.

---

# Technology

Framer Motion

Material UI transitions

CSS transitions for simple interactions.

---

# Accessibility

Animations should

- Never trigger seizures
- Avoid flashing
- Respect reduced motion
- Maintain focus order
